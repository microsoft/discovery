# Example input files

`smiles.csv` - 10 reference compounds spanning the GWP range (Novec 649 ~1, HFC-23 ~14600, SF6 ~24300) with published GWP100 + lifetime values from IPCC AR6, Hodnebrog 2020, and 3M datasheets. Used for batch-prediction acceptance tests in Phase 4.

The batch action consumes any CSV with a SMILES column (default name: `smiles`).
