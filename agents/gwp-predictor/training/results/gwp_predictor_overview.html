Training Set Chemistry Classes

GWP Distribution

The training set spans 8 orders of magnitude in GWP100 (0.003 to 26,700), with the heaviest concentration in the 100-10,000 range (HFCs, CFCs, HCFCs). The holdout set covers a similar range but with a different chemistry composition (Tanimoto-clustered to ensure scaffold novelty).
